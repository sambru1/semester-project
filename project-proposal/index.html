Project Proposal

Overview

I want to make a website for a bubble tea stand/ restaurant. This establishment doesn’t actually exist, so I can have a little bit of freedom with how I make it and what I add onto it. I will be using images from copyright free websites like “unsplash” and “pexels, and also illustrate elements as I move along with the project. The feel of the project will be light, colourful and fun.

CSS elements

I’m not sure what we will be learning for the rest of the semester, so I won’t know what to add for until we learn them. Some things I want to have one my page is:

  • Drop down selection from nav bar.
  • Hover buttons that display special animations.
  • A carousel of pictures on the main page.
  • A menu, about us, and location page.
  • A logo that might include an animation.
  • Linkws socials on the navigation bar.
  • It will be responsive for large screens and small screens like iphones.
  • I will have the nav bar stick to the top of the screen so users will always be able to direct themselves around.

User Persona & Color palette

Bubble tea usually attracts a younger crowd between the ages of 12-25. I wanna make my page look fun and poppy when users come to use it. The colors I have chosen are more on the lighter and pop side. They are subject to change as I move on with my project, but for now I want to go with these.
